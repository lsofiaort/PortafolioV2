# Portafolio Lizzie Ortiz - Vite + LESS

Portafolio profesional desarrollado con Vite y estilos generados desde LESS.

## Requisitos de la tarea

- Uso de web bundler Vite.
- CSS generado por LESS.
- Archivos minificados en producción mediante `npm run build`.
- CSS purgado a solo las reglas utilizadas con PurgeCSS.
- Despliegue compatible con GitHub Pages.

## Comandos

```bash
npm install
npm run dev
npm run build
npm run preview
```
